# fair-game

Given match stats, estimate a "fair" scoreline and compare it to the actual result.

## structure
- src/: scripts (fetch data, build dataset, train model, generate reports)
- fair_game/: package code
- data/: ignored (raw, interim, processed)
- reports/: markdown report plus figures

## quick start
1. add your api key to .env
2. fetch data
3. train model
4. generate report
